Cloud Security Automation

About This Project

This Terraform script automates the deployment of a secure and scalable cloud environment on AWS. It provisions essential resources such as VPCs, subnets, security groups, and EC2 instances to create a functional infrastructure for a web server.

Key Features

  • Automated deployment of a Virtual Private Cloud (VPC).
  • Creation of public subnets and an Internet Gateway.
  • Launch of an EC2 instance with SSH and HTTP access.
  • Integration of a security group to enforce access policies.

Terraform Code Snippet

# Example from the script
resource "aws_instance" "web_server" {
  ami           = "ami-12345678"
  instance_type = "t2.micro"
  subnet_id     = aws_subnet.public.id
  key_name      = "your-key-pair"
  tags = {
    Name = "WebServer"
  }
}
