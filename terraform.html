Cloud Security Automation

About This Project

This Terraform script mirrors the structure of my cybersecurity home lab, automating the deployment of resources such as a VPC, EC2 instances for the web server and Nessus server, and an S3 bucket for centralized logging. It demonstrates a secure, scalable, and practical cloud environment designed for learning and showcasing cloud security automation.

Key Features

  • Automated deployment of a Virtual Private Cloud (VPC) and subnets.
  • Provisioning of EC2 instances for the web server and Nessus vulnerability scanner.
  • Creation of an S3 bucket for centralized log storage and integration with other tools.
  • Security group configurations to enforce controlled access.

Terraform Code Snippet

# Example from the script
resource "aws_instance" "web_server" {
  ami           = "ami-12345678"
  instance_type = "t2.micro"
  subnet_id     = aws_subnet.public.id
  key_name      = "your-key-pair"
  tags = {
    Name = "WebServer"
  }
}
